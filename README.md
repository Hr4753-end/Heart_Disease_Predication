# Heart_Disease_Predication
🎯 Overview
This project implements a Logistic Regression model to predict the presence of heart disease in patients based on medical indicators. The model achieves ~84% accuracy and includes advanced evaluation techniques such as ROC curves, cross-validation, and feature importance analysis.
Why This Project?

✅ Production-Ready: Professional ML pipeline with comprehensive metrics
✅ Educational: Learn data science concepts through real implementation
✅ Visual: 9 different visualization types for result interpretation
✅ Scalable: Can be adapted for other medical prediction tasks


✨ Features
Core Machine Learning

🤖 Logistic Regression - Efficient binary classification algorithm
📊 5-Fold Cross-Validation - Robust model evaluation
🎯 Feature Scaling - StandardScaler for optimal performance
🔄 Train-Test Split - 80/20 data stratification

Advanced Evaluation Metrics

📈 ROC Curve with AUC Score - Model discrimination ability
🎯 Precision & Recall - Detailed classification breakdown
📊 Confusion Matrix - TP, TN, FP, FN visualization
📉 Calibration Curve - Probability reliability assessment
🔍 Feature Importance - Coefficient-based feature ranking
⚖️ F1-Score - Balanced performance metric
